# amd-performance-build
Documentation and benchmarking of an all-AMD, no-RGB high-performance gaming PC.

All-AMD, no-RGB gaming PC documented as a system: component choices, configuration,
benchmarks, thermals, and iteration over time.

## Goals
- High 1440p/4K gaming performance
- Good airflow and stable thermals
- Clean, no-RGB configuration
- Measured changes with before/after data

## Current Configuration
- CPU: Ryzen 7 7800x3d -- Purchased
- GPU: XFX Swift 9070 XT -- Purchased
- Motherboard: MSI MAG B850 TOMAHAWK MAX WIFI AM5 - Not Purchased 
- Memory: G.SKILL Flare X5 DDR5 32GB 6000MT/s CL30 - Not Purchased
- Storage: WD BLACK SN7100 2TB - Not Purchased
- Power Supply: be quiet! Pure Power 13 M 1000W - Not Purchased
- Case: H9 Flow (2025) -- Purchased
- Case Fans: Arctic P14 Pro Fans -- Purchased
- CPU Cooling: Arctic Liquid Freezer Pro 420mm - Not Purchased
- OS: Windows 11 Home - Not Purchased

## Methodology
Benchmarks are to be run on a clean system with documented settings.
Temperatures will be measured under sustained load.
